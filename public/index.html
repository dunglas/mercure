    Authorization Type
    Authorization
    mercureAuthorization

    The authorization cookie is set automatically by the server during discovery.
    Note: The SSE connection uses the auth type selected at the moment of subscription. Changing this setting will not affect active connections until you resubscribe.

    JWT Algorithm

    HS256: Symmetric shared secret. RS256: Asymmetric RSA keys (PEM or JWKS).

    Hub config required: The hub accepts only one key type at a time. To use HS256, ensure publisher_jwt/subscriber_jwt uses a symmetric secret (not a PEM key or JWKS URL).

    Required to publish or to subscribe to private updates.
