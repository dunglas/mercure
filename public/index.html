Debugging Tools

Subscribe

One topic per line. Each line is interpreted with the matcher type picked above. Use * to subscribe to every topic.

Examples:

Not subscribed.

Publish

One IRI or string per line: one update is published per topic.

Settings

Will be filled automatically if you discover a resource.

The authorization cookie will be set automatically by the server only if you discover a demo endpoint.

Required to publish, or to subscribe to private updates.
Claim structure to use: 

{
  "mercure": {
    "subscribe": [
      { "match": "<topic or pattern>", "match_type": "exact|urlpattern" }
      // use "*" for all, omit subscribe for public only
    ],
    "publish": [
      { "match": "<topic or pattern>", "match_type": "exact|urlpattern" }
      // use "*" for all, omit publish to not allow publishing
    ]
  }
}

match_type defaults to Exact when omitted.
